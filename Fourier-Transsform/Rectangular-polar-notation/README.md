## Rectangular to Polar-Notation

coverts given(waveform.c) rectangular signal's to polar notation.

```
gcc rectangular2polar.c waveforms.c -lm -o Rect-polar-notation
./Rect-polar-notation
```

#### for plotting:
```
gnuplot gnu-script.p
```
